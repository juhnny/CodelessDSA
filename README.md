## 2 선형 데이터 구조

### 스택
**스택**은 추가된 요소를 사용 가능한 메모리의 가장 앞 주소에 배치하는 선형 데이터 구조의 한 종류

스택에 요소를 추가하는 동작을 **푸시**push, 삭제하는 동작을 **팝**pop이라고 함. 삭제 시에는 마지막으로 추가된 요소를 삭제. 스택 구조를 **후입선출**First in first out, LIFO 데이터 구조라고 함.
